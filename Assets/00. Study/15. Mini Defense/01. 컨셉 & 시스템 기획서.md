# 미니 디펜스 — 컨셉 & 시스템 요약 기획서

---

## 1. 개요

| 항목 | 내용 |
|---|---|
| 제목(가제) | 미니 디펜스 (Mini Defense) |
| 장르 | 2D 탑뷰 타워 디펜스 |
| 플랫폼 | PC (Windows) |
| 시점 / 조작 | 탑다운 그리드, 마우스 클릭 중심 |
| 테마 | SF 우주 정거장 |
| 한 줄 컨셉 | 정해진 길로 진군하는 적을, 타일에 유닛을 배치해 막아내는 웨이브 방어 게임 |

---

## 2. 목표와 승패 조건

- **목표**: 3개의 웨이브에 걸쳐 몰려오는 적이 목표 지점에 도달하지 못하도록 방어한다.
- **승리**: 마지막(3번째) 웨이브의 모든 적을 처치한다.
- **패배**: 기본 체력이 0이 된다. (적이 목표 지점에 도달할 때마다 감소)

---

## 3. 핵심 게임플레이 루프

```
준비(유닛 배치/업그레이드)
  → [다음 웨이브] 호출
  → 적 진군 + 유닛 자동 전투
  → 적 처치로 자원 획득
  → 웨이브 종료(클리어 보상)
  → 다시 준비 …
  → 3웨이브 클리어(승리) 또는 체력 0(패배)
```

---

## 4. 맵과 경로

- 사각 타일 그리드(기준 10 x 8).
- 타일 종류
  | 타일 | 설명 |
  |---|---|
  | 건설 패드 | 유닛 배치 가능 |
  | 통로 | 적 이동 경로. 배치 불가 |
  | 스폰 타일 | 적이 등장. 배치 불가 |
  | 목표 타일 | 적의 도착점. 배치 불가 |
- 적은 스폰 타일에서 **고정된 경로(웨이포인트)** 를 따라 목표 타일까지 이동한다. (경로는 배치와 무관하게 고정)

---

## 5. 유닛 (3종)

공통 규칙
- 배치되면 사거리 안의 적을 **자동으로** 공격한다.
- 모든 공격은 **크리티컬 확률**을 가진다(확률 발동 시 배율 피해).
- **3레벨**까지 업그레이드 가능(레벨마다 비용과 성능 상승).

| 유닛 | 역할 | 공격 방식 | 특징 |
|---|---|---|---|
| 저격수 | 단일 처리 | 한 대상에게 투사체 | 조준 방식(선두/최근접 등)을 가짐. 고피해 |
| 포격병 | 다수 처리 | 착탄 반경 광역 타격 | 한 번에 여러 적(최대 대상 수 제한) |
| 지휘관 | 지원 | 약한 자체 공격 + 주변 아군 강화 아우라 | 공격 유닛만 강화(지휘관끼리는 강화 안 함) |

### 조준 방식
저격수/포격병은 사거리 안 여러 적 중 하나를 고르는 기준을 가진다: 선두 / 후미 / 최근접 / 최대체력 / 최소체력.

### 유닛 초기 스탯 (레벨 1)
| 유닛 | 소환비용 | 피해 | 사거리 | 공격간격(초) | 크리확률 | 비고 |
|---|---|---|---|---|---|---|
| 저격수 | 100 | 20 | 3.0 | 1.0 | 20% | 기본 조준: 선두 |
| 포격병 | 150 | 12 | 2.0 | 1.5 | 15% | 착탄 반경 1.0, 최대 4대상 |
| 지휘관 | 120 | 8 | 2.0 | 1.3 | 10% | 버프 반경 2.0, 공격력 +30%·공속 +20% |

### 업그레이드 (레벨 2 / 3)
| 유닛 | Lv2 비용 / 주요 상승 | Lv3 비용 / 주요 상승 |
|---|---|---|
| 저격수 | 80 / 피해 34·크리 25% | 160 / 피해 55·사거리 3.5·크리 30% |
| 포격병 | 120 / 피해 20 | 240 / 피해 32·사거리 2.5·크리 20% |
| 지휘관 | 100 / 버프 +45%·+25% | 200 / 버프 +60%·+30%·사거리 2.5 |

| 적 | 최대체력 | 이동속도 | 방어 | 저항 | 처치 보상 | 도달 시 체력 감소 |

## 7. 웨이브

- 총 **3웨이브**. 플레이어가 **[다음 웨이브] 버튼으로 직접 시작**(수동 호출).
- **조기 호출 보너스**: 준비 시간을 남기고 일찍 호출하면 남은 시간에 비례해 자원 보너스(초당 2).
- 웨이브 전멸 시 **클리어 보상** 지급.

| 웨이브 | 구성 | 클리어 보상 |
|---|---|---|
| W1 | 보병 8 (0.8초 간격) | 50 |
| W2 | 정찰병 6 (0.6초) + 보병 4 (0.8초, 5초 뒤 시작) | 70 |
| W3 | 중장갑 8 (1.0초) + 파괴자 1 (10초 뒤 등장) | 150 |

## 8. 경제

- **단일 자원: 광물.** 시작 보유 300.
- **획득**: 적 처치 보상 + 웨이브 클리어 보상 + 조기 호출 보너스.
- **소비**: 유닛 소환 / 업그레이드.
- **판매**: 유닛 제거 시 그동안 투자한 금액의 **70% 환불**.

---

## 9. 전투 규칙

피해는 다음 순서로 계산된다.

```
기본 피해
  → 크리티컬: 확률 발동 시 피해 x 배율(기본 2.0)   [공격 유닛 기준]
  → 방어: 피해 - 방어값(고정 감산, 최소 0)          [맞는 적 기준]
  → 저항: 피해 x (1 - 저항%)                         [맞는 적 기준]
= 최종 피해
```

- 크리티컬 확률은 유닛/레벨별로 다르다.
- 속성 상성은 없다.

## 10. 조작 & UX

| 입력 | 대상 | 결과 |
|---|---|---|
| 좌클릭 | 빈 건설 패드 | 유닛 생성 메뉴 표시 → 선택 시 소환(광물 소비) |
| 좌클릭 | 아군 유닛 | 관리 메뉴 표시 → 업그레이드 또는 판매 |
| 좌클릭 | 통로/스폰/목표/점유 타일 | 배치 불가(피드백) |
| 마우스 호버 | 아군 유닛 | 정보 팝업(이름·레벨·현재 스탯·설명) |

- 배치/호버 시 **사거리 표시**.
- 정보 팝업의 스탯은 **기본값 + 버프**를 함께 보여준다(예: 20 (+6) = 26).

## 11. 저장 / 이어하기

- 저장은 **웨이브 사이에서만** 가능하다(웨이브 진행 중에는 잠금).
- 저장 항목: 현재 웨이브 번호 · 보유 광물 · 남은 체력 · 배치된 유닛(종류·레벨·타일 좌표).
- 이어하기: 저장 시점의 준비 상태로 복원한다.
